## 어딨쥐? (Whereizit?)

> **"필요할 때 바로 그곳에!"**
> 지도 위에서 생활 편의시설(화장실, 쓰레기통, 흡연구역, 정수기)의 위치를 쉽게 찾고 공유하는 앱

---

## 📱 앱 소개

'어딨쥐?'는 일상생활에서 갑자기 필요해지는 다양한 **생활 편의시설의 위치를 공유하는 지도 기반 플랫폼**입니다.

사용자는 지도 위에서 **개방 화장실, 쓰레기통, 흡연구역, 정수기** 등 다양한 편의시설의 위치를 등록하거나, 다른 사용자들이 공유한 위치 정보를 실시간으로 확인할 수 있습니다.

이를 통해 급할 때 편의시설을 찾아 헤매는 불편함을 해소하고, 모두가 더 편리하고 쾌적한 도시 생활을 누릴 수 있도록 돕는 것을 목표로 합니다.

---

## 주요 기능

* 지도에서 현재 위치 확인 및 주변 편의시설 마커 표시
* 새로운 편의시설 위치 등록 (이름, 설명, **시설 유형(화장실, 쓰레기통 등) 태그** 포함)
* 등록된 편의시설 정보(개방 시간, 청결도 등) 공유 및 확인
* 주변 편의시설 목록 확인 (현재 위치에서의 거리 포함)


---

## 기술 스택

* Swift
* UIKit + FlexLayout + PinLayout
* Firebase Firestore
* Naver Maps SDK

---

### 앞으로 개발할 기능

* 사용자가 보고 있는 화면에 있는 마커만 주변 편의시설 목록에 보여주기
* 편의 시설에 대한 **별점 평가** 기능
* 편의 시설 위치에 댓글 기능
* **길 찾기 연동** 기능 (선택한 편의시설로 바로 이동)
* **시설 유형별 필터링** 기능 (예: 화장실만 보기, 쓰레기통만 보기)
